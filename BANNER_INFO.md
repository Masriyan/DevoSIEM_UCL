# Banner Image Information

## Required Banner Image

**Filename:** `devosiem_banner.png`

**Location:** Root directory of the repository

**Recommended Specifications:**
- **Dimensions**: 1200px Ã— 400px (3:1 aspect ratio)
- **Format**: PNG with transparency support
- **File size**: < 500KB (optimized for GitHub)
- **Content suggestions**:
  - Project title: "DEVO SIEM Use Case Library"
  - Subtitle: "Production-Ready Detection Rules"
  - Visual elements: Security/SIEM themed graphics
  - Color scheme: Professional security colors (blues, grays, accent colors)
  - Include shield/security icons
  - MITRE ATT&CK badge/mention
  - "45 Use Cases" badge

## Design Guidelines

### Text Elements
- **Main Title**: "DEVO SIEM Use Case Library"
- **Tagline**: "Production-Ready Detection Rules & Response Playbooks"
- **Stats Badge**: "45 Use Cases | 13 Categories | MITRE ATT&CK Mapped"

### Visual Style
- Modern, professional design
- Security/cybersecurity themed
- Not too busy - keep it clean
- Good contrast for readability on both light and dark GitHub themes

### Branding Considerations
- **DO NOT** use official DEVO Technology logos or branding
- **DO NOT** imply official affiliation with DEVO
- **DO** make it clear this is a community project
- **DO** use generic security/SIEM imagery
